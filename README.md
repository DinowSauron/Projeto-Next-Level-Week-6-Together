## Tecnologias Utilizadas:

* React 
* Yarn 
* Firebase 
    * Autenticação
    * Banco de dados em tempo real




## Comandos Utilizados

* ``yarn create react-app letmeask --template typescript``
* ``cd letmeask``
* ``yarn add firebase``
* ``yarn start``
* ``yarn add node-sass@^5.0.0``
* ``yarn add react-router-dom``
* ``yarn add @types/react-router-dom -D``
